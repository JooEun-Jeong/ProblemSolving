### 🔹 1. **단일 시작점 최단 경로 문제**
#### ✅ 유형 설명  
- 특정 정점에서 다른 모든 정점까지의 최단 경로를 구하는 전형적인 다익스트라 문제.  
- **그래프는 가중치가 양수**여야 함.  

#### 🛠 활용 방법  
1. 우선순위 큐(Priority Queue, Min-Heap) 또는 `deque`를 사용해 최적화.  
2. 시작 정점에서의 거리를 `0`, 나머지 정점은 `INF`로 설정.  
3. 방문하지 않은 정점 중 최단 거리인 정점을 선택 후 인접 노드 업데이트.  
4. 최단 거리 테이블 갱신하면서 반복.  

#### 📌 예제 문제  
- 백준 1753번: **최단경로**  
- LeetCode 743: **Network Delay Time**

#### ✅ 유형 설명  
- 출발점에서 도착점까지의 최단 경로만 필요.  
- 모든 정점이 아니라 특정 목적지까지 도달하는 것이 목표.  

#### 🛠 활용 방법  
1. **목표 지점이 정해져 있으므로** 해당 지점에서 다익스트라 탐색 종료 조건을 추가해 최적화.  
2. 만약 **출발점이 여러 개라면**, 각 출발점에서 다익스트라를 수행하고 가장 짧은 경로를 찾기.  
3. 그래프가 희소하면 **우선순위 큐 사용**하여 `O(E log V)`로 효율적으로 해결 가능.  

#### 📌 예제 문제  
- 백준 1916번: **최소비용 구하기**  
- LeetCode 787: **Cheapest Flights Within K Stops**

### 8. **📌 0-1 BFS를 사용해야 하는 경우**
0-1 BFS는 **그래프에서 간선 가중치가 0 또는 1인 경우** 최단 경로를 빠르게 찾을 때 사용합니다.

#### **✅ 0-1 BFS를 적용할 조건**
1️⃣ **그래프의 가중치가 0 또는 1인 경우**
   - 예제: 이동 비용이 0 또는 1인 미로 탐색
   - 백준 1261번 **알고스팟 문제**처럼 벽을 부수면 1, 아니면 0인 경우

2️⃣ **최단 경로를 구하는 문제**
   - 출발점에서 도착점까지 이동할 때 **가장 적은 비용**을 구하는 문제

3️⃣ **BFS를 기반으로 하되, 가중치를 고려해야 하는 경우**
   - 일반적인 BFS는 간선 가중치가 동일할 때만 사용 가능
   - 다익스트라는 모든 가중치가 양수일 때 사용 가능
   - **0과 1이 혼합된 경우 0-1 BFS가 가장 효율적**

#### **🚀 0-1 BFS를 사용해야 하는 대표적인 문제 유형**
1. **미로 탈출 (가중치가 0과 1)**  
   - 벽을 부수지 않으면 가중치 0, 부수면 가중치 1  
   - **예제: 백준 1261번 (알고스팟)**

2. **도로 건설 문제**  
   - 도로 건설 시 비용이 0 또는 1  
   - 예: 비용 없이 이동할 수 있는 도로와, 1의 비용이 드는 도로가 섞여 있음  

3. **텔레포트 문제**  
   - 순간이동(비용 0)과 일반 이동(비용 1)이 섞여 있는 경우  

4. **로봇 이동 문제**  
   - 특정 조건을 만족하면 이동 비용 0, 그렇지 않으면 1
